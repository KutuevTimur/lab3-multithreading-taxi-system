## Многопоточная система беспилотных такси (Лабораторная работа №3)

Этот проект — модель работы системы беспилотных такси, выполненная в рамках лабораторной работы по Java.  
В программе реализована многопоточность с использованием `Thread`, `Runnable`, `BlockingQueue` и механизмов синхронизации.


### Функионал
- Генератор клиентов создаёт случайные заявки на поездку.  
- Заявки попадают в потокобезопасную очередь (`BlockingQueue`).  
- Диспетчер забирает заявки из очереди, ищет ближайшее свободное такси и назначает его.  
- Каждое такси работает в отдельном потоке:
  - ждёт заказ,  
  - «едет» к клиенту,  
  - «везёт» клиента в нужную точку,  
  - сообщает диспетчеру о завершении поездки.  

Программа запускает симуляцию на фиксированное время, затем корректно завершает работу всех потоков.
- Main.java — запуск симуляции
- Taxi.java — логика работы такси (поток)
- Dispatcher.java — распределение заказов
- ClientGenerator.java — генератор клиентских заявок
- RideRequest.java — модель заявки
- Point.java — координаты
- DistanceUtil.java — расчёт расстояний
- TaxiStatus.java — статус такси

### Как запустить
1. Открыть проект в **IntelliJ IDEA**.  
2. Запустить файл **Main.java**.  
3. В консоли увидеть работу потоков такси, диспетчера и генератора клиентов.
